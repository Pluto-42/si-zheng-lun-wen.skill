# Current Source Refresh Protocol

Use this protocol whenever the user asks for a current red-course paper, or whenever the paper uses laws, plans, meetings, policy reports, public figures, or annual targets.

Current baseline in this draft: 2026-05-31.

## Refresh Order

1. Identify the paper topic and course.
2. Check whether any example is time-sensitive:
   - laws and regulations
   - five-year plans
   - national conferences or plenary sessions
   - government work reports
   - annual targets
   - current technology, ecological, education, or youth policy
3. Search official or primary sources first:
   - 中国政府网: https://www.gov.cn/
   - 中国人大网: https://www.npc.gov.cn/
   - 国家发展改革委: https://www.ndrc.gov.cn/
   - 教育部: https://www.moe.gov.cn/
   - 生态环境部: https://www.mee.gov.cn/
   - 新华网 / 人民网 official reposts when primary text is unavailable.
4. Record the exact title, date, and URL before using the example.
5. Replace stale examples using `theme-taxonomy-and-refresh.md`.

## Do Not Use As Current Without Refresh

- "抗疫英雄" paragraphs from 2020-2022.
- "十九大报告指出" as present-stage framing.
- "全面建成小康社会" as a future task.
- Old "互联网+" examples when the topic is AI, digital governance, or new quality productive forces.
- Old exam-prediction pages that say "今年热点" unless the year matches the current task.

## Safe Wording When Not Fully Verified

If a source cannot be verified during the task, avoid exact dates and titles. Use cautious wording:

- "近年相关政策持续强调..."
- "当前政策语境中..."
- "以最新官方文件为准，可从...角度展开"

Do not invent a full citation.

## Suggested Update Log Format

When adding new current examples to the material library, use:

```markdown
### Example title

- Added/checked: YYYY-MM-DD.
- Source: title, date, URL.
- Theme fit:
- Course fit:
- Use as:
- Avoid:
```

## Maintenance Rule

Before each major use of this skill after a date change, update the "Current 2025/2026 Example Pool" if:

- a newer government work report is available,
- a newer five-year plan document or implementation interpretation appears,
- new laws relevant to market, education, ecology, AI, or youth are passed,
- official language around a theme changes,
- the user explicitly asks for the latest examples.
