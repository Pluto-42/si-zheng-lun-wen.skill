# Argument Patterns

Use these structures before using style replacements. Pick one main skeleton per paper.

## Pattern 1: Concept Clarification

Use for 马克思主义基本原理 and theory-heavy papers.

Structure:

1. State a common misunderstanding.
2. Define the key concept.
3. Explain the internal contradiction or mechanism.
4. Connect to a historical or contemporary case.
5. Conclude with theoretical and practical implications.

Example paragraph move:

审视这一问题，不能只停留在表层判断。更关键的是弄清楚：该概念在马克思主义理论中指向的并不是孤立现象，而是一组历史关系。由此观之，问题的重点不在于简单肯定或否定，而在于说明它在何种条件下发挥作用，又在何种条件下显露界限。

## Pattern 2: Historical Choice

Use for 中国近现代史纲要.

Structure:

1. Begin with a historical dilemma.
2. Show why old paths failed or had limits.
3. Explain how a new political force, theory, or road emerged.
4. Connect historical choice with present confidence.
5. Avoid triumphalism; let evidence carry the conclusion.

Example paragraph move:

近代中国的道路选择，并不是抽象理念之间的纸面比较，而是在民族危亡、社会动荡和人民苦难中反复试错的结果。正因为如此，历史结论才不是凭空得来，而是在一次次实践检验中形成。

## Pattern 3: Contemporary Governance Logic

Use for 中国特色社会主义.

Structure:

1. Name a real governance problem.
2. Explain why market, society, state, technology, or law alone cannot solve it.
3. Introduce institutional arrangement or policy logic.
4. Show concrete effect and remaining tension.
5. Conclude with "发展与规范", "效率与公平", "活力与秩序", or another pair of real tensions.

Example paragraph move:

这一安排的意义，不在于多增加一个政策口号，而在于把分散的社会力量纳入可预期、可协调的制度框架之中。它既释放活力，也划定边界；既回应现实压力，也为长期治理留下空间。

## Pattern 4: Youth Growth And Responsibility

Use for 思想道德与法治 and youth-themed essays.

Structure:

1. Start from a campus, family, volunteer, career, or media observation.
2. Identify the value conflict behind it.
3. Use course concept to analyze the conflict.
4. Move from personal choice to public responsibility.
5. End with concrete action, not only elevated feeling.

Example paragraph move:

青年担当并不只存在于宏大叙事之中。它也体现在一次认真完成的实践、一种面对困难时不轻易退后的选择，以及把个人理想放进社会需要中重新校准的过程里。

## Logic Templates From User Images

Use sparingly and adapt to the user's topic.

### Phenomenon Critique

审视当下，[某现象]屡见不鲜。毋庸讳言，其产生有复杂的社会根源。但问题的症结在于，我们往往只看到了表面，而忽略了[本质]。有鉴于此，我们亟需[提出对策]。

Use for openings or sub-claims. Replace `[某现象]`, `[本质]`, and `[提出对策]` with concrete content.

### Dialectical Analysis

固然，[对方观点]有其合理之处。然而，倘若我们仅仅止步于此，便容易陷入[片面性]。更进一步说，真正的[核心论点]，应当是[你的观点]。

Use in middle paragraphs when correcting a one-sided view.

### Concluding Elevation

一言以蔽之，[重述中心论点]。唯有[行动/态度]，方能[美好愿景]。这，既是时代的要求，亦是我们青年一代的应有之义。

Use only when the essay already has a youth/practice dimension. Otherwise write a quieter conclusion.
